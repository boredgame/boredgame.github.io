Upcoming Games

I downloaded and tested the following games, so we can start playing basically any time we like.

I am using a framework called Tabletop Simulator which is basically just a physics engine designed for boardgames. There is an amazing modding community supporting this framework, which meanes after you have the engine, almost all of the more commonly known boardgames are downloadable for free!

And this means of course that we can also create our own games pretty easily.
